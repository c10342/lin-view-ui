# 国际化

[[toc]]

---

`lin-view-ui` 组件内部默认使用中文，若希望使用其他语言，则需要进行多语言设置。以英文为例，在 main.js 中：

```javascript
// 完整引入 lin-view-ui
import Vue from "vue";
import LinVueUi from "lin-view-ui";
import enUS from "lin-view-ui/lib/lang/en-US.js";

Vue.use(LinVueUi, { locale: enUS });
```

目前 `lin-view-ui` 内置了以下语言：

- 简体中文（zh-CN）
- 英语（en-US）
