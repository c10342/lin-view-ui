# 介绍

---

`lin-view-ui` 是一款基于 `Vue.js 2.0` 的前端 UI 组件库，主要集成了我平时在开发中使用到的 UI 组件

## 特性

- 基于 `Vue` 开发的 UI 组件
- 使用 npm + webpack + babel 的工作流，支持 ES2015
- 提供友好的 API，可灵活的使用组件
- 支持按需引入，减少项目打包体积
- 偏向于业务组件
- 提供完善的文档

## 浏览器支持

- 现代浏览器和 IE10 及以上
- [Electron](http://electron.atom.io/)
