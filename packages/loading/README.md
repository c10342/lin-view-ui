# Loading


## 在线文档

[http://ui.linjiafu.top/loading](http://ui.linjiafu.top/loading)


## 安装

```
npm install @lin-view-ui/loading
```

## 快速开始

```javascript
import Vue from 'vue';
import Loading from '@lin-view-ui/loading';
import '@lin-view-ui/loading/dist/style.css'

Vue.use(Loading);
```

## 浏览器支持

- 现代浏览器和 IE10 及以上
- [Electron](http://electron.atom.io/)
